# AriA — WhatsApp RPG Dungeon Bot

> *Enter the dungeon. Earn your rank. Survive.*

AriA is a feature-rich RPG bot built for WhatsApp groups using the
[Baileys](https://github.com/WhiskeySockets/Baileys) library. Players
register as Hunters, choose a class, grind stats, explore procedural
dungeons, and battle monsters — all through WhatsApp chat commands.

## ⚔️ Features

- Hunter System — Register, level up, and rank from F → S
- Class Builds — Tank, Mage, Assassin and more, each with unique skill trees
- Dungeon Exploration — Enter dungeons, step through rooms, and encounter monsters
- Skill Combat — Turn-based fights using learnable moves
- Dungeon Market — Rotating shop with tiered weapons, potions & skill scrolls
- Economy — Earn coins, trade items, and climb the ranks
- Daily Rewards — Log in daily for free gold

## 🛠 Built With

- [Baileys](https://github.com/WhiskeySockets/Baileys) — Lightweight WhatsApp Web API
- Node.js
- Hand-coded architecture (minimal AI assistance 😤)

## 🚀 Commands

| Command | Description |
|---|---|
| !awaken | Register as a Hunter |
| !me | View your stats & rank |
| !enter | Enter a dungeon |
| !onward | Explore / find monsters |
| !shop | Browse the market |
| !daily | Claim daily coins |
So much more
